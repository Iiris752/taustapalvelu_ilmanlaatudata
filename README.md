Tämä projekti tarjoaa rajapinnan tietokantaan tallennetun ilmanlaatudatan hakemiseen ja analysointiin

- Yhden päivän mittaustulosten haku valitulle mittausasemalle
- Mittausten lukumäärän haku
- Sensorikohtaisen päivittäisen keskiarvon laskenta
- Datan yhdistäminen (measurement + sensor) SQL-kyselyillä

Teknologiat
- Python
- PostgreSQL
- SQL
- Flask
